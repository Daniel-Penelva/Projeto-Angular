O que é angular?

Um framework com base em linguagem JavaScript;
Utilizado para criação de SAPs;
Possui arquitetura baseada em componentes;
Muito utilizado em aplicções de grande porte;
Compõe o Stack MEAN
Angular 2+ é uma versão diferente do Angular JS (clássico);
Estrutura de organização do projeto bem definida, pormove simplicidade e reaproveitamento de código;
Sistema de tooling muito rico: CLI, modulo de HTTP, router e outros;
Integração nativa com TypeScript;
Mantido pelo Google;

O que vai precisar?

Node.js: para instalar o angular, outros pacotes e rodar scripts;
Um editor de código - por exemplo, VS code
Abra o terminal Git Bash Here;
Instalar o CLI do angular com: npm install - g @angular/cli
No terminal utilizando "ng" mostra os comandos disponiveis que podem ser executados;
No terminal coloque o caminho onde você quer que abra o projeto;
Agora é possível criar projetos utilizando: ng new nome_projeto
No terminal utilizando "ls" mostra o projeto disponível;
No terminal utilizando "cd" marca o projeto disponível;
No terminal utilizando ".code" abre o VS Code;
Para abrir o terminal no VS Code - com o botao direito do mouse na aba do projeto vai em "open in integrated Terminal";
Para rodar a aplicação vai no terminal utilizando "ng serve";
Pode-se alterar os arquivos dentro da pasta app para refletir mudanças no navegador;
